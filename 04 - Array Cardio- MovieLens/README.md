## 💡 About the Project

I built this while practicing JavaScript array methods.

Instead of just doing exercises in the console, I wanted to build something **real** using the same concepts — so every feature in this app is powered by a native array method.

---

## 🧠 What Each Array Method Does

| Method | Feature in the App |
|---|---|
| `.filter()` | Search by title, filter by genre & decade |
| `.sort()` | Sort by rating, year, or A→Z |
| `.reduce()` | Calculate avg rating, avg year, top genre |
| `.map()` | Render every movie card to the DOM |

---

## ✨ Features

- 🔍 Real-time search by movie title
- 🎭 Filter by genre
- 📅 Filter by decade (1980s → 2020s)
- ↕️ Sort by rating, year, or title
- 📊 Live stats — average rating, top genre, average year
- 🎨 Clean dark UI with movie poster cards

## 🛠️ Built With

## 📚 What I Learned

- How `.filter()` can chain multiple conditions in one pass
- How `.reduce()` can build objects (not just sum numbers)
- How `.map()` can return DOM elements, not just data
- How to combine all four methods to build a real interactive UI
